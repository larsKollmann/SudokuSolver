Sudoku-Löser 1.0
================

Hallo lieber Kunde.
Schön, dass Sie sich für den Sudoku-Löser der SWE Gruppe 2 (PO 2015) entschieden haben.




1. Was ist der Sudoku-Löser
===========================

Der Sudoku-Löser ist ein Programm um Sudokus zu generieren, lösen, 
speichern (exportieren) und laden (importieren).
Diese Aktionen sind für die Sudoku-Größen 9x9 (Der bekannte Standart), 16x16, 25x25 und 36x36 möglich.




2. Was benötige ich
===================

Um den Sudoku-Löser zu starten brauchen Sie eine Java-Virtual-Machine mit der Version 8 auf Ihrem System
installiert und korrekt konfiguriert
(Aktuell unter http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html.
Falls dieser Link nichtmehr funktioniert, hilft eine Internetrecherche nach "download JRE 8").
Neuere Versionen könnten auch funktionieren, werden von unserem Kundensupport aber nicht unterstützt.




3. Wie funktioiert das Programm
===============================

Nachdem Sie alle Anforderungen aus Punkt 2 erfüllen machen Sie einen Doppelklick auf die Datei
"Sudoku.jar". Ganz oben in dem Programmfenster sehen Sie einen Knopf "Hilfe".
Wenn Sie auf "Hilfe" und dann "Hilfe anzeigen" klicken, wird Ihnen erklärt,
wie das Programm funktioniert.




4. Importieren / Laden
======================

Falls Sie ein Sudoku manuell erstellen wollen, um dieses in das Programm zu importieren, 
müssen Sie einige Regeln beachten.
Es werden 3 Dateiformate unterstützt.
Für die einzelnen Dateiformate gibt es sogenannte howtos.
- howtoCSV.txt: Erklärt wie eine CSV-Datei erstellt wird
- howtoJSON.txt: Erklärt wie eine JSON-Datei erstellt wird
- howtoXML.txt: Erklärt wie eine XML-Datei erstellt wird


Viel Spass beim "Lösen" von Sudokus ;)
