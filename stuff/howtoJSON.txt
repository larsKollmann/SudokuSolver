Die .json Datei sieht folgendermassen aus:


!Start json Datei

{ 
	"size": $N,
	"sudoku": [
		$Sudoku
	]
}

!Ende Json Datei
  
$N kann dabei die Werte 9, 16, 25 oder 36 annehmen.
Alle anderen führen zu Fehlern.

$Sudoku steht für eine Aneinanderreihung von Zahlen zwischen 0 und N.
Zahlen sind durch ein Komma (,) getrennt.
0 Steht für ein leeres Feld. Eine zahl von 1 bis N steht für
den jeweiligen Wert.

Diese Zahlen werden spaltenweise eingelesen.
Die erste Zahl ist die oben links, die 2. rechts daneben und so weiter.

Wenn N Zahlen eingelesen worden sind, beginnt automatisch
eine neue Zeile.
Zur Übersichtlichkeit sollten Sie die einzelnen Zeilen in
ihrer .json Datei durch Zeilenumbrüche trennen.

Die Anzahl Zahlen in $Sudoku muss $N mal $N sein.
Also im Beispiel von $N = 9 müssen es 81 Zahlen sein,
ansonsten kann die Datei nicht importiert werden.


Eine komplette Datei mit einem 9x9 Sudoku gibt es in der Datei "beispiel_9x9.json".
Eine Datei mit einem 16x16 Sudoku in der Datei "beispiel_16x16.json".
